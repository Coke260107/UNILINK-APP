# 기능 정리

# 사용자 관련 상세 정보

## 프로필 생성

- 닉네임 -> 중복안됨
- 성별 (비공개)
- MBTI (비공개)
- 연령대 (비공개) [10대, 20대, ...]
- 프로필 이미지
- 간단한 소개글 (선택)
- 현재 위치 정보 (GPS 사용 현재 내 위치)

## 신고

- 신고 종류
    1. 모임 신고
    2. 게시물 신고
    3. 채팅 신고
    4. 게시물 댓글 신고

### 모임 신고

- 정책
    1. **신고 카테고리** 설정과 **신고 사유**를 입력하여 신고
    2. 한 모임에 대해 한 번만 신고 가능

### 게시물 신고

- 정책
    - 한 게시물에 대해 한 번만 신고 가능

### 채팅 신고

- 정책
    - 같은 모임에 있는 한 사람에 대해 한 번 신고 가능

### 게시물 댓글 신고

- 정책
    - 한 댓글에 대해 한 번만 신고 가능

# 모임 관련 상세 정보

## 모임

### 모임 생성

- 모임 제목
- 모임 카테고리
- 모임 상세 소개
- 모집 마감 날짜
- 모임 D-Day + 시간
- 모임 만료 시간 = 모임 D-Day 날자로부터 하루 후

### 모임 수정

- 모임 제목
- 모임 카테고리
- 모임 상세 소개
- 모임 D-Day + 시간
- 모임 만료 시간
- 수정 가능 시간 = 모임 D-Day 날짜로부터 하루 전
- 모임 장속 선택

### 모임 참가

- 정책
    1. 사용자는 모임의 모임 마감 기간 전까지 자유롭게 참여할 수 있으며 모임의 방장은 참여자를 강제 퇴장 시킬 수 있다.

### 모임 검색

- 모임 검색 종류
    1. 모임 제목으로 검색
    2. 모임 검색 페이지 들어왔을 때 전체 모임 표시
    3. 카테고리 + 검색어

### 모임 삭제

- 정책
    - 모임 삭제는 모임 참여자의 과반 이상의 결정으로 삭제 가능

### 모임 나가기

- 정책
    - 모임 나가기는 모집 마감 날짜 전까지 가능

# 게시판 관련 상세 정보

- 게시판 종류
    - 자유게시판
    - 모임 홍보 게시판

## 모임 홍보 게시판

- 정책
    - 홍보 게시판은 사용자가 속한 모임에 한에서 작성 가능
    - 자신 작성한 홍보 게시물 모임 모집 마감 날짜 전까지 생성, 수정 가능

## 자유 게시판

- 정책
    - 자신이 작성한 게시물은 언제든지 수정 가능

## 채팅

- 기능
    - 채팅 보낸 시간 조회
    - 채팅 참여한 멤버 조회
    - 참여한 멤버의 프로필 조회
    - 사용자의 닉네임으로 참여
    - 읽음/안읽음 표시
    - 메지지 공감/취소 (좋아요)
        - 공감한 유저 목록 조회
    - 채팅 고정 (공지)
    - 채팅 검색
    - 삭제 투표

- 정책
    - 참여자는 모임 참가와 동시에 채팅방 이용 가능
    - 참여자가 모임에 퇴장했을 때 채팅방도 탈퇴된다
    - 사용자가 퇴장해도 채팅 유지
    - 채팅방 탈퇴시 모임 자동 탈퇴

## 마이페이지

- 프로필 조회 및 수정
- 일정 조회
- 참여한 모임 목록
    - 참여한 모임에 대한 정보 조회
    - 모임 정보 팝업에서 채팅방 이동 가능
- 로그아웃 및 탈퇴 가능
- 작성한 게시물 및 댓글 조회
- 게시글/모임 좋아요 목록
- 신고 당한 횟수에 따른 점수 조회(색깔로 표시) - 보류

## 알림

- 채팅 수신 알림
- 모임 D-day 알림
    - 하루전과 당일에 알림
- 좋아요/댓글 알림
- 모임 삭제에 대한 투표 알림
- 모임 삭제 알림
- 알림함에서 알림 목록 조회
